## 📖 About The Project

"Flores y Corazones" (Flowers and Hearts) is an advanced front-end web development experiment that pushes the boundaries of what is possible with pure CSS art and animation. This project creates a highly immersive, romantic, and calming nighttime landscape without relying on heavy JavaScript libraries, WebGL, or external image assets. 

Every visual element you see—from the gently swaying grass and the glowing flower petals to the intricately layered starry night sky and floating heart bubbles—is meticulously crafted using CSS shapes, gradients, and keyframe animations. It serves as a testament to the power of modern CSS3 in creating dynamic, visual storytelling on the web.

## ✨ Key Features

* **🎨 100% Pure CSS Art:** No external images or textures were used. The entire scene is painted using CSS properties.
* **🌌 Immersive Night Sky:** A deep, atmospheric background created entirely with stacked `radial-gradient` and `repeating-linear-gradient` techniques to simulate stars and night lighting.
* **🌷 Dynamic Blooming Animations:** Smooth, synchronized, and beautifully delayed animations for the growing stems, blooming leaves, and glowing elements using `@keyframes` and CSS custom properties (`--d`).
* **💖 Floating Heart Bubbles:** Continuous, multi-layered burst animations of floating hearts, structured using inline SVGs and complex CSS `transform` properties.
* **⚡ Ultra-Lightweight & Performant:** Employs minimal Vanilla JavaScript solely to remove a class on window load, ensuring all the heavy lifting is handled efficiently by the browser's native CSS rendering engine.

## 🛠️ Technologies Used

* **HTML5:** Semantic structuring and inline SVG integration for scalable, crisp heart icons.
* **CSS3:** * Advanced `@keyframes` animations and dynamic transitions.
  * CSS Variables (Custom Properties) for timing and color management.
  * `clip-path` and `mask-image` for organic leaf and grass shapes.
  * Complex layered `box-shadow` and multi-directional gradients for depth and glow effects.
* **Vanilla JavaScript:** A single-line script to handle the initial window load event and trigger the animation sequence perfectly without lag.

## 📂 File Structure

* `index.html` - The core markup containing the DOM structure, SVG paths, and layout containers.
* `style.css` - The heart of the project. Contains all the styling, artistic drawing logic, and complex animation keyframes.
* `script.js` - A tiny script that acts as the catalyst to start the animations once the DOM is fully ready.
